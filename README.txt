Для запуска приложения Вам потребуется:
1) Ввести в поле response = requests.get('http://0.0.0.0:5000/logs?') нужный IP адрес машины вместо 0.0.0.0
2) Запустить приложение.
3) В приложении реализованы кнопки чтения и вывода данных в приложение, обновление данных и фильтрация данных по самому новому запросу.
